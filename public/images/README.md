# Imagens do Projeto

Esta pasta contém as imagens estáticas do site:
- Coloque aqui a foto profissional da equipe (ex: `paula.jpg`).
- Coloque aqui a logo da empresa (ex: `logo.png` ou `logo.svg`).

As imagens inseridas aqui podem ser acessadas diretamente no site pelo caminho absoluto, por exemplo: `<img src="/images/paula.jpg" />`.
